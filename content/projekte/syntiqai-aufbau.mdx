## 24. März 2026 — Kernprodukt definiert

Der wichtigste Tag bisher: das Produkt bekommt eine klare Form.

**Kernprodukt: Private AI Workspace**

Ein DSGVO-konformes, selbst gehostetes Multi-Agent System für KMUs. Jeder Mitarbeiter bekommt einen persönlichen KI-Agenten — erreichbar via WhatsApp oder Telegram. Der Agent kennt das Unternehmen, die Kollegen, die Prozesse.

**Zielgruppe:** Nicht-technische KMUs in Österreich — Steuerberater, Arztpraxen, Fahrschulen.

**USP gegenüber Microsoft Copilot:** Datensouveränität von Grund auf, kein SharePoint-Datenleck, RBAC eingebaut.

**Pilot-Partner gefunden:** techflow IT-Solutions (Flo Koller) — zwei Nutzer, echter Praxistest.

**Tech-Entscheidungen:**
- LLM Phase 1: Azure OpenAI (EU-Datenspeicherung, DSGVO-konform, pay-as-you-go)
- LLM Phase 2: Ollama + Mistral lokal (maximale Datensouveränität)
- Infrastruktur: Docker auf Proxmox VM → später Azure Container Apps
